A simple and fun Python console game where the player has to guess a randomly chosen number between 1 and 100. The game provides hints whether the guess should be higher or lower, and counts the number of attempts.

---

## Features

- Randomly generates a number between 1 and 100.
- Provides hints for higher or lower guesses.
- Counts the number of attempts.
- Fun and interactive console game experience.

---

## How to Play

1. Clone this repository or download the `perfect_guess.py` file.
2. Run the game using Python:
3. Follow the instructions on the console:
    Guess the number.
    If your guess is too high, the game will ask for a lower number.
    If your guess is too low, the game will ask for a higher number.
4. Keep guessing until you find the correct number!
5. The game will congratulate you and show how many attempts you took.
